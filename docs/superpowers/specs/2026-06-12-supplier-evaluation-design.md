# 공급업체 평가 관리 시스템 설계 문서

**작성일**: 2026-06-12  
**대상**: 와이어로프 제조업체 구매팀  
**형태**: 단일 HTML 파일 (localStorage 기반)

---

## 1. 개요

공급업체 재평가 보고서를 디지털화하고, 업체별 평가 이력 관리·출력·내보내기를 지원하는 단일 HTML 웹 앱.

---

## 2. 화면 구성 (4개 뷰)

### 2-1. 대시보드
- 요약 카드: 전체 업체 수 / 승인 업체 수 / 재평가 임박(D-30 이내)
- 재평가 예정 목록 테이블 (D-30: 빨강, D-60: 주황, D-90: 노랑)
- 등급별 현황 (A/B/C/D 개수)

### 2-2. 업체 관리
- 업체 목록 테이블 (검색, 카테고리 필터)
- 업체 등록·수정·삭제 모달 폼
- 필드: 업체명, 품목카테고리, 사업자번호, 주소, 담당자, 연락처, Critical 여부, 등록일, 비고

### 2-3. 평가 입력
공급업체 재평가 보고서 양식과 동일한 레이아웃:

| 섹션 | 내용 |
|------|------|
| 헤더 | 평가서 No. (자동채번), 공급업체명 (드롭다운), 평가일자 |
| Risk 평가 | Critical Supplier 선택 시 활성화. 납기/품질 관리 항목별 발생도·심각도·취약성 입력, Risk 평가 결과 |
| 평가항목 | ISO 9001 보유 여부 + 유효기간, Self Assessment, Supplier Chain 검증, 제품 평가 |
| 전년도 성과 | 납기율 목표/실적, 부적합율 목표/실적 → 점수 실시간 자동계산 |
| 총점/등급 | 자동계산 및 표시 |
| 참고사항 | 7개 고정 텍스트 (표준 문구) |
| 평가 결과 | A/B/C/D 자동 선택, 승인/승인불가 자동 표시 |
| 결재란 | 평가자, 검토자, 승인자 텍스트 입력 (직전 평가에서 자동불러오기) |

**자동입력**: 업체 선택 시 해당 업체의 직전 평가 데이터 전체 자동 채움 → 변경 항목만 수정

### 2-4. 평가 이력
- 업체별 전체 평가 이력 타임라인
- 이력 클릭 → 보고서 미리보기
- PDF 출력 / 수정 / 삭제

## 4. 점수 계산 로직

### ISO 9001 (50점)
- 보유 + 유효기간 내: **50점**
- 미보유 또는 만료: **0점**

### 납기율 (25점)
```
score = Math.min((실적 / 목표) * 25, 25)
```

### 부적합율 (25점)
- 목표 = 0%인 경우: 실적 0% → 25점, 실적 > 0% → 0점
- 목표 > 0%인 경우: `score = Math.max((1 - 실적/목표) * 25, 0)`

### 등급 기준
| 등급 | 총점 | 결과 | 재평가 주기 |
|------|------|------|------------|
| A | 90 ~ 100 | 승인 | 3년 |
| B | 70 ~ 89  | 승인 | 2년 |
| C | 50 ~ 69  | 조건부 승인 | 1년 |
| D | 0 ~ 49   | 승인불가 | 즉시 재평가 |

## 5. 출력 및 내보내기

| 기능 | 방식 |
|------|------|
| PDF 출력 | `window.print()` + 인쇄 전용 CSS (보고서 레이아웃 그대로) |
| Excel 내보내기 | SheetJS(CDN) 사용, 평가이력 전체 또는 단건 |
| CSV 내보내기 | 순수 JS, 평가이력 전체 |
| JSON 백업/복원 | 전체 localStorage 데이터 파일로 저장·불러오기 |

---

## 6. 기술 스택

- **HTML/CSS/JS**: 순수 바닐라 (프레임워크 없음)
- **SheetJS**: CDN (Excel 내보내기용, 오프라인 시 CSV 대체)
- **저장소**: localStorage (`suppliers`, `evaluations` 키)
- **배포**: 단일 `.html` 파일, 더블클릭 실행

---

## 7. 자동입력 흐름

```
평가입력 화면 → 업체 선택
    ↓
해당 업체의 최근 평가 기록 조회
    ↓
전체 항목 자동 채움 (ISO 유효기간, 결재자명 등 포함)
    ↓
사용자가 변경 항목만 수정
    ↓
저장 → 새 평가 레코드 생성 (기존 기록 유지)
```

## 8. 재평가 알림 기준

- **D-30 이내**: 빨강 강조 + 대시보드 최상단 노출
- **D-60 이내**: 주황 강조
- **D-90 이내**: 노랑 강조
- **초과**: 회색 (정상)
