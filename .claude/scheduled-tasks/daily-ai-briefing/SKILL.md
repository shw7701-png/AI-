매일 아침 LLM과 생성AI 위주로 AI 동향 브리핑을 만들어라.
**수신자는 와이어로프 업체 구매 담당자**이므로, 구매·조달·공급망·제조 업무와 연관성이 높은 뉴스를 상단에 배치한다.

## 실행 절차

### 1단계: 뉴스 수집 (news-collector 에이전트에 위임)

`news-collector` 에이전트에 다음을 요청한다:
- 당일 날짜 기준 LLM·생성AI 뉴스 수집
- 구매·조달·공급망 관련 뉴스를 1순위로 검색
- 각 뉴스에 업무 연관성 등급(🔴/🟡/⚪) 포함하여 반환

### 2단계: 뉴스 정렬 및 HTML 브리핑 생성

수집된 뉴스를 **업무 연관성 순**으로 정렬 후 `show_widget`으로 렌더링한다.

브리핑 구성:
1. **오늘의 핵심 뉴스** — 연관성 등급 배지 표시, 🔴 핵심 뉴스 최상단
2. **구매 담당자 Action Point** — 오늘 브리핑에서 실무에 바로 적용 가능한 인사이트 2~3줄
3. **주목 트렌드** — 3~4가지 칩 형태
4. **주목할 도구 & 서비스** — 연관성 등급 포함
5. **오늘의 추천 읽을거리** — 심층 아티클 2편

### 3단계: 저장 및 커밋 (git-publisher 에이전트에 위임)

`git-publisher` 에이전트에 다음을 요청한다:
- 생성된 HTML을 `briefings/YYYY-MM-DD.html`로 저장
- git commit & push 실행

## 출력 형식 요구사항

- HTML 위젯으로 출력 (`show_widget` 사용)
- 상단에 날짜와 "AI 모닝 브리핑" 제목 + "구매 담당자 Edition" 부제
- "LLM & 생성AI 특화" 태그 + "와이어로프·조달" 태그 표시
- 각 뉴스 카드에 업무 연관성 배지(🔴/🟡/⚪) 및 한 줄 이유 표시
- **구매 담당자 Action Point 섹션** 필수 포함
- 하단에 "다음 브리핑 — 내일 오전 7:00 KST" 표시
- 심화 질문을 유도하는 `sendPrompt` 버튼 2~3개 (구매 업무 관련)
- 다크모드 호환 CSS 변수 사용
- 언어: 한국어 요약 + 원문 출처 링크
