매일 아침 LLM과 생성AI 위주로 AI 동향 브리핑을 만들어라. 아래 순서로 실행하라.

## 실행 절차

### 1단계: 뉴스 수집 (news-collector 에이전트에 위임)

`news-collector` 에이전트에 다음을 요청한다:
- 당일 날짜 기준 LLM·생성AI 최신 뉴스 수집
- Top 3 뉴스, 주목 트렌드, 주목 도구, 추천 읽을거리 반환

### 2단계: HTML 브리핑 생성

수집된 뉴스를 바탕으로 `show_widget`으로 HTML 브리핑을 렌더링한다.

브리핑 구성:
1. **오늘의 핵심 뉴스 Top 3** — 제목, 2~3문장 요약, 출처 링크
2. **주목 트렌드** — 3~4가지 칩 형태 표시
3. **주목할 도구 & 서비스** — 이름, 한 줄 설명, 상태 태그
4. **오늘의 추천 읽을거리** — 제목, 출처, 링크

### 3단계: 저장 및 커밋 (git-publisher 에이전트에 위임)

`git-publisher` 에이전트에 다음을 요청한다:
- 생성된 HTML을 `briefings/YYYY-MM-DD.html`로 저장
- git commit & push 실행

## 출력 형식 요구사항

- HTML 위젯으로 출력 (`show_widget` 사용)
- 상단에 날짜와 "AI 모닝 브리핑" 제목 표시
- "LLM & 생성AI 특화" 태그 표시
- 각 뉴스 카드에 왼쪽 색상 보더(`border-left`) 적용
- 하단에 "다음 브리핑 — 내일 오전 7:00 KST" 표시
- 관련 심화 질문을 유도하는 `sendPrompt` 버튼 2~3개 포함
- 다크모드 호환 CSS 변수 사용
- 언어: 한국어 요약 + 원문 출처 링크(영문 포함)
